Destek

Günümüz iletişim ve bilgi dünyasında, kurumsal ve kişisel verilerin korunması oldukça önem arz etmektedir. Bilgilerin korunmasını sağlayan en temel ölçüt ise şifrelendirme yöntemi. Fakat şifrelendirmenin de yeterli olmadığı sonuçlar ortaya çıkabilmekte. Bu noktada devreye bazı güvenlik tedbirleri giriyor. Veri güvenliğini sağlamak da işte bu önlemlerden birisidir. Verilerin dışarıdan gelebilecek tüm erişimlere ve siber saldırılara karşı korunmasını ifade eder.

SIKÇA SORULAN SORULAR

ZERDAVA File Shredder cihazlarıma nasıl kurabilirim?

ZERDAVA File Shredder'ı Windows tabanlı cihazlarınıza kurabilmek için gerekli kurulum paketlerini bilgisayarınıza indirmeniz gerekmektedir. ZERDAVA'nın resmi web sitesi üzerinden ürün grupları hakkında bilgi alabilir ve istediğiniz ürün için deneme sürümü kurulum paketini ücretsiz indirebilirsiniz

ZERDAVA File Shredder'ın Android tabanlı cihazlarınıza kurabilmek için Google Play uygulama mağazasını ziyaret edebilirsiniz. Google Play üzerinden ZERDAVA'yı cihazınıza kurmak için burayı tıklayınız.

ZERDAVA File Shredder'ın desteklediği işletim sistemleri nelerdir?

ZERDAVA File Shredder tarafından desteklenen işletim sistemleri:

Microsoft Windows

Windows 10, Mobile ve IoT Core Sürümü hariç, x86 tabanlı 32-bit veya 64-bit

Windows 8.1, RT ve Başlangıç Sürümü hariç, x86 tabanlı 32-bit veya 64-bit

Windows 8, RT ve Başlangıç Sürümü hariç, x86 tabanlı 32-bit veya 64-bit

Android 4.1 (Jelly Bean) ve üstü, x86 veya ARM tabanlı 32-bit veya 64-bit

ZERDAVA File Shredder ücretsiz mi?

ZERDAVA File Shredder, Windows tabanlı cihazlar için ücretli olmasına rağmen 7 gün boyunca istediğiniz cihazda ücretsiz deneme imkanı sunulur. Deneme süreci için herhangi bir üyelik, kredi kartı bilgisi vb. bilgilere ihtiyaç duyulmaz. 7 gün sonunda tüm tüm fonksiyonları çalışmaya devam eder. Sadece bazı algoritmaları kullanamaz ve ürün güncelleştirmelerini alamazsınız.

ZERDAVA File Shredder'ı satın almadan önce kullanabilir miyim?

ZERDAVA File Shredder, Android ve Windows cihazlar için 7 gün boyunca kullanabileceğiniz deneme sürümüne sahiptir

ZERDAVA File Shredder, Android cihazımda neden dosyalarıma erişim izni istiyor?

Güvenli veri silme uygulamaları diğer güvenli veri silme uygulamaları gibi dosya veya boş alan silme işlemi yapabilmek için dosyalarınıza erişebilmesi gerekir.

Windows platformu için çevrimiçi ve çevrimdışı kurulum paketleri arasındaki fark nedir?

ZERDAVA File Shredder, Windows'un farklı sürümlerindeki yapısal değişiklikler ve desteklenen işlemci mimarileri nedeniyle farklı kurulum paketlerine sahiptir.

Çevrimiçi (Online) kurulum paketleri işletim sistemini ve işlemci mimarisini otomatik olarak tanır ve gereken doğru kurulum paketini sizin için indirerek kurulumu gerçekleştirir. Eğer Windows işletim sisteminizin sürümü ve işlemci mimariniz hakkında bir fikriniz yoksa ve aktif İnternet bağlantınız varsa çevrimiçi kurulum paketlerini kullanmanız tavsiye edilir.

Çevrimdışı (Offline) kurulum paketleri doğrudan bir Windows sürümü ve işlemci mimarisine uygun olarak hazırlanmıştır. Kurulum için herhangi bir İnternet bağlantısı gerektirmez. Ayrıca kurulum paketleri indirdiğiniz zamana kadarki tüm güncelleştirmeleri içerir. Çevrimdışı kurulum paketleri, çok sayıda bilgisayara kurulum yapmak veya İnternet bağlantısının olmadığı cihazlara kurulum yapmak isteyenlere tavsiye edilir.
